Gerador de Código de Acesso

Ferramenta interna — use apenas você, o vendedor. Gere um código para cada comprador, copie o HASH gerado e cole no arquivo data/access-codes.js. Envie o CÓDIGO (não o hash) para o comprador.

Dica: inclua algo identificável no código (nome ou data da venda) pra você lembrar depois quem é quem — o hash em si não é reversível, então anote em algum lugar seu (ex: uma nota no celular) qual código foi vendido pra qual pessoa.
